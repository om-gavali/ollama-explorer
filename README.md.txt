# Ollama Explorer

A full-stack web app to interact with local LLMs using Ollama.

## Tech Stack
Frontend: React
Backend: Spring Boot
LLM Runtime: Ollama

## Features
- Chat with local models
- Model selection
- API integration with Ollama
- Clean UI

## Project Structure
frontend/ → React app
backend/ → Spring Boot API

## Run locally

### Backend
cd backend
mvn spring-boot:run

### Frontend
cd frontend
npm install
npm start
